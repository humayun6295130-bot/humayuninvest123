# **App Name**: AscendFolio

## Core Features:

- Portfolio Overview Dashboard: Displays key metrics such as total portfolio value, daily/overall gains and losses, and a quick summary of top holdings.
- Asset Management: Allows users to manually add, edit, and remove specific investment assets (stocks, ETFs, cryptocurrencies) from their personal portfolio.
- Real-time Pricing & Charting: Provides up-to-date pricing data and basic historical charts for individual assets tracked within the user's portfolio.
- User Authentication: Secure user registration, login, and profile management for personalized portfolio tracking.

## Style Guidelines:

- Primary color: A sophisticated, deep indigo blue (#334C99), chosen to convey trust and stability without being overly conventional, forming the cornerstone of the application's visual identity.
- Background color: A very light, desaturated blue (#EBEFF6), offering a clean and professional canvas that visually connects to the primary color while ensuring readability.
- Accent color: A vibrant yet harmonious sky blue (#52BBDB), providing a dynamic contrast for interactive elements and highlights, complementing the primary palette effectively.
- Body and headline font: 'Inter' (sans-serif), selected for its modern, clean, and highly readable characteristics, suitable for conveying financial data clearly.
- Minimalist and clean financial-themed icons, prioritizing clarity and quick recognition of investment concepts.
- A structured, responsive dashboard layout focusing on data hierarchy and intuitive navigation for an efficient user experience.
- Subtle and smooth transitions for data updates and view changes to enhance user perception of real-time information and fluidity.
